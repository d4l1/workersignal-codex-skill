# Contributing

Thank you for helping improve the WorkerSignal AI connector skill.

## Rules

- Do not commit private prompts, private code, credentials, client names, user data, or project details.
- Do not include examples copied from real users unless every detail is anonymized and approved.
- Open a pull request for proposed changes.
- The repository owner reviews and approves merges.

## Suggested Changes

Good pull requests usually improve:

- Redaction guidance.
- Safer signal examples.
- Connector consent language.
- Claude, Cursor, Codex, and Agent Skills compatibility notes.
- Documentation clarity.

Do not add integrations that depend on private or unofficial APIs.
